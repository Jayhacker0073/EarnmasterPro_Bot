## Setup

1. Clone this repository
2. Set environment variables:
   - BOT_TOKEN: Your Telegram bot token
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the bot:
   ```bash
   python bot.py
   ```
